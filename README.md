# LoHa Autobrowser — Releases

Release binaries cho [LoHa Autobrowser](https://loha-license-server.vercel.app) — desktop app tự động hoá Facebook profile management.

Source code: PRIVATE repo (không public).

## Download

Phiên bản mới nhất: xem [Releases](https://github.com/longloha/loha-autobrowser-releases/releases/latest).

File installer: `LoHa.Autobrowser_<version>_x64-setup.exe` (Windows 10/11 x64).

## Auto-update

App tự động check update qua `latest.json` ở GitHub Releases. User không cần download thủ công sau lần cài đầu.
